DHCP4Java

The free open-source DHCP API and Server for Java.

DHCP4Java is a free open-source 100% Java DHCP API and Server. It covers three domains: a general purpose DHCP API (suitable for clients, servers, relays...), a DHCP Server framework (similar to HTTP Servlets) and a DHCP static Server (coming soon). A full featured dynamic server is for future plans.

Introduction

This open-source project provides 3 levels of functionalities:

  1. API for DHCP: a general purpose for receiving and sending DHCP/BOOTP packets with support for a variety of options with a high-level datatypes support. Note: it is limited to IPv4 DHCP.
  2. Server Framework: a general purpose framework based on a model similar to the well-known HTTP Servlet model. It allows rapid development of servers with vary specific behaviours.
  3. DHCP Server: (coming soon) a generic DHCP Server. First version is a static server, dynamic server is for future plans.
