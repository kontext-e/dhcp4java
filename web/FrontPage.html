DHCPCluster - dhcp4java

The free open-source DHCP Server and DHCP API for Java.

dhcp4java is a free open-source 100% Java DHCP Server: general purpose DHCP server ranging from a simple LAN server to a wide range WAN server handling millions customers. A general pupose DHCP API is also available as a stand-alone library, suitable for client/server/relay development.

Link to the dhcp4java sourceforge project.

Introduction

This open-source project provides 3 levels of functionalities:

  1. DHCP Server: a generic DHCP Server. [to be detailed]

  2. API for DHCP: a general purpose API for receiving and sending DHCP/BOOTP packets with support for a variety of options with a high-level datatypes support. It includes a server framework based on a model similar to the well-known Servlet model. Note: it is limited to IPv4 DHCP.
